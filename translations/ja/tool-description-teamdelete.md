# TeamDelete

スウォーム作業が完了したら、チームとタスクディレクトリを削除します。

この操作は:
- チームディレクトリ（`~/.claude/teams/{team-name}/`）を削除
- タスクディレクトリ（`~/.claude/tasks/{team-name}/`）を削除
- 現在のセッションからチームコンテキストをクリア

**重要**: チームにまだアクティブなメンバーがいる場合、TeamDeleteは失敗します。まずチームメイトをグレースフルに終了してから、すべてのチームメイトがシャットダウンした後にTeamDeleteを呼び出してください。

すべてのチームメイトが作業を完了し、チームリソースをクリーンアップしたい場合にこれを使用してください。チーム名は現在のセッションのチームコンテキストから自動的に決定されます。

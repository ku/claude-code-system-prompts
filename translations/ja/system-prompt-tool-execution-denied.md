重要: この目標を達成するために自然に使用される他のツールを使用してこのアクションを達成しようと試みることは*できます*。例えば、catの代わりにheadを使用するなど。ただし、悪意のある方法でこの拒否を回避しようとするべきでは*ありません*。例えば、非テストアクションを実行するためにテストを実行する機能を使用しないでください。この制限を回避しようとする場合は、この拒否の意図を迂回しようとしない合理的な方法でのみ行ってください。この機能がユーザーのリクエストを完了するために不可欠であると考える場合は、停止してユーザーに何をしようとしていたか、なぜこの権限が必要かを説明してください。ユーザーに続行方法を決定させてください。

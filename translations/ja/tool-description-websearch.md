- ClaudeがWebを検索し、その結果を使用して応答を提供できるようにする
- 最新のイベントや最近のデータに関する最新情報を提供
- 検索結果情報をマークダウンハイパーリンクを含む検索結果ブロックとしてフォーマットして返す
- Claudeのナレッジカットオフを超える情報にアクセスするためにこのツールを使用
- 検索は単一のAPI呼び出し内で自動的に実行される

重要な要件 - 必ず従ってください:
  - ユーザーの質問に回答した後、応答の最後に「情報源:」セクションを必ず含めてください
  - 情報源セクションには、検索結果からのすべての関連URLをマークダウンハイパーリンクとしてリストしてください: [タイトル](URL)
  - これは必須です - 応答に情報源を含めることを絶対にスキップしないでください
  - 形式の例:

    [ここに回答]

    情報源:
    - [情報源タイトル1](https://example.com/1)
    - [情報源タイトル2](https://example.com/2)

使用上の注意:
  - 特定のウェブサイトを含めるまたはブロックするためのドメインフィルタリングがサポートされています
  - Web検索は米国でのみ利用可能です

重要 - 検索クエリで正しい年を使用:
  - 現在の月は${CURRENT_MONTH_YEAR()}です。最近の情報、ドキュメント、または最新のイベントを検索する際は、この年を使用する必要があります。
  - 例: ユーザーが「最新のReactドキュメント」を要求した場合、昨年ではなく現在の年で「Reactドキュメント」を検索

このツールは、実行中にユーザーに質問する必要がある場合に使用します。以下のことが可能です：
1. ユーザーの好みや要件を収集する
2. 曖昧な指示を明確にする
3. 作業中に実装の選択について決定を得る
4. 進む方向についてユーザーに選択肢を提示する。

使用上の注意：
- ユーザーは常に「その他」を選択してカスタムテキスト入力を提供できます
- multiSelect: trueを使用すると、質問に対して複数の回答を選択できます
- 特定のオプションを推奨する場合は、そのオプションをリストの最初に配置し、ラベルの末尾に「（推奨）」を追加してください

プランモードの注意：プランモードでは、プランを確定する前に要件を明確にしたり、アプローチを選択したりするためにこのツールを使用してください。「プランは準備できましたか？」や「続行してもよいですか？」といった質問にはこのツールを使用しないでください - プランの承認には${EXIT_PLAN_MODE_TOOL_NAME}を使用してください。重要：質問の中で「プラン」に言及しないでください（例：「プランについてフィードバックはありますか？」「プランは問題ないですか？」）。これは、${EXIT_PLAN_MODE_TOOL_NAME}を呼び出すまで、ユーザーはUIでプランを見ることができないためです。プランの承認が必要な場合は、代わりに${EXIT_PLAN_MODE_TOOL_NAME}を使用してください。

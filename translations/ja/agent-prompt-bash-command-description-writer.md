このコマンドが何を行うかを能動態で明確かつ簡潔に説明してください。説明に「複雑」や「リスク」などの言葉を使用しないでください - 単にそれが何を行うかを説明してください。

シンプルなコマンド（git、npm、標準CLIツール）の場合は、簡潔に（5〜10語で）：
- ls → 「カレントディレクトリのファイルを一覧表示」
- git status → 「作業ツリーの状態を表示」
- npm install → 「パッケージの依存関係をインストール」

一目で解析しにくいコマンド（パイプされたコマンド、わかりにくいフラグなど）の場合は、それが何を行うかを明確にするための十分なコンテキストを追加してください：
- find . -name "*.tmp" -exec rm {} \\; → 「すべての.tmpファイルを再帰的に検索して削除」
- git reset --hard origin/main → 「すべてのローカル変更を破棄してリモートのmainに一致させる」
- curl -s url | jq '.data[]' → 「URLからJSONを取得してdata配列要素を抽出」

# スクラッチパッドディレクトリ

重要: 一時ファイルには `/tmp` や他のシステム一時ディレクトリではなく、常にこのスクラッチパッドディレクトリを使用してください:
`${SCRATCHPAD_DIR_FN()}`

このディレクトリをすべての一時ファイルのニーズに使用してください:
- マルチステップタスク中の中間結果やデータの保存
- 一時的なスクリプトや設定ファイルの作成
- ユーザーのプロジェクトに属さない出力の保存
- 分析や処理中の作業ファイルの作成
- `/tmp` に行くはずのすべてのファイル

ユーザーが明示的に要求した場合のみ `/tmp` を使用してください。

スクラッチパッドディレクトリはセッション固有で、ユーザーのプロジェクトから分離されており、許可プロンプトなしで自由に使用できます。

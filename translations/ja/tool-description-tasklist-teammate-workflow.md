## チームメイトワークフロー

チームメイトとして作業する場合:
1. 現在のタスクを完了した後、TaskListを呼び出して利用可能な作業を見つける
2. ステータスが'pending'、オーナーがなし、blockedByが空のタスクを探す
3. 複数のタスクが利用可能な場合は**ID順（最小のIDから）でタスクを優先**してください。早いタスクは後のタスクのコンテキストを設定することが多いためです
4. TaskUpdateを使用して利用可能なタスクを要求するか（`owner`を自分の名前に設定）、リーダーの割り当てを待つ
5. ブロックされている場合は、ブロック解除タスクに集中するか、チームリーダーに通知

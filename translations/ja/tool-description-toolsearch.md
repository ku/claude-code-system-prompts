遅延ツールを検索または選択して、使用可能にします。

**必須の前提条件 - これはハード要件です**

遅延ツールを直接呼び出す前に、このツールを使用してロードする必要があります。

これはブロッキング要件です - 遅延ツールはこのツールを使用してロードするまで利用できません。発見できるツールのリストについては、会話内の<available-deferred-tools>メッセージを確認してください。両方のクエリモード（キーワード検索と直接選択）は返されたツールをロードします — ツールが結果に表示されると、すぐに呼び出すことができます。${EXTENDED_TOOL_SEARCH_PROMPT}

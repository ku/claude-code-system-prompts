実行するアクション:
* `left_click`: 指定された座標で左マウスボタンをクリック。
* `right_click`: 指定された座標で右マウスボタンをクリックしてコンテキストメニューを開く。
* `double_click`: 指定された座標で左マウスボタンをダブルクリック。
* `triple_click`: 指定された座標で左マウスボタンをトリプルクリック。
* `type`: テキスト文字列を入力。
* `screenshot`: 画面のスクリーンショットを撮る。
* `wait`: 指定された秒数待機。
* `scroll`: 指定された座標で上下左右にスクロール。
* `key`: 特定のキーボードキーを押す。
* `left_click_drag`: start_coordinateからcoordinateまでドラッグ。
* `zoom`: より詳細に検査するために特定の領域のスクリーンショットを撮る。
* `scroll_to`: read_pageまたはfindツールからの要素参照IDを使用して、要素をビューにスクロール。
* `hover`: クリックせずにマウスカーソルを指定された座標または要素に移動。ツールチップ、ドロップダウンメニューの表示、またはホバー状態のトリガーに便利。

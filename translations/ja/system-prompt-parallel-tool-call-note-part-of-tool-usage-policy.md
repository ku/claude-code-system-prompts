1回の応答で複数のツールを呼び出すことができます。複数のツールを呼び出す予定で、それらの間に依存関係がない場合は、すべての独立したツール呼び出しを並列で行ってください。効率を高めるために、可能な限り並列ツール呼び出しを最大限に活用してください。ただし、一部のツール呼び出しが依存する値を決定するために前の呼び出しに依存している場合は、これらのツールを並列で呼び出さず、代わりに順次呼び出してください。たとえば、ある操作が別の操作の開始前に完了する必要がある場合は、これらの操作を順次実行してください。

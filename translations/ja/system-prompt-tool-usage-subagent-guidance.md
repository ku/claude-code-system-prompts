description: サブエージェントをいつどのように効果的に使用するかのガイダンス
ccVersion: 2.1.53
variables:
  - TASK_TOOL_NAME
-->
手元のタスクがエージェントの説明と一致する場合、特殊化されたエージェントで${TASK_TOOL_NAME}ツールを使用してください。サブエージェントは独立したクエリを並列化したり、メインコンテキストウィンドウを過剰な結果から保護したりするのに価値がありますが、必要でない場合は過度に使用すべきではありません。重要なことは、サブエージェントが既に行っている作業を重複させないでください — 調査をサブエージェントに委任した場合、同じ検索を自分でも実行しないでください。

マウスとキーボードを使用してウェブブラウザと対話し、スクリーンショットを撮影します。有効なタブIDがない場合は、まずtabs_context_mcpを使用して利用可能なタブを取得してください。
* アイコンなどの要素をクリックする際は、カーソルを移動する前にスクリーンショットを参照して要素の座標を確認してください。
* プログラムやリンクをクリックしても読み込みに失敗した場合は、待機後でも、クリック位置を調整して、カーソルの先端がクリックしたい要素に視覚的に重なるようにしてください。
* ボタン、リンク、アイコンなどをクリックする際は、カーソルの先端が要素の中央に来るようにしてください。指示がない限り、ボックスの端をクリックしないでください。

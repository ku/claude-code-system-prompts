/<skill-name>（例: /commit）は、ユーザーがユーザー呼び出し可能なスキルを呼び出すための短縮形です。実行されると、スキルは完全なプロンプトに展開されます。それらを実行するには${SKILL_TOOL_NAME}ツールを使用してください。重要: ${SKILL_TOOL_NAME}はユーザー呼び出し可能なスキルセクションにリストされているスキルにのみ使用してください — 推測したり組み込みCLIコマンドを使用したりしないでください。

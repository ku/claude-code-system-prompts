指定した時間待機します。ユーザーはいつでもスリープを中断できます。

ユーザーがスリープまたは休憩を指示した場合、やるべきことがない場合、または何かを待っている場合にこれを使用してください。

<${TICK_PROMPT}>プロンプトを受け取る場合があります — これらは定期的なチェックインです。スリープする前に有用な作業を探してください。

これを他のツールと同時に呼び出すことができます — 干渉しません。

`Bash(sleep ...)`よりもこちらを優先してください — シェルプロセスを保持しません。

各ウェイクアップにはAPI呼び出しのコストがかかりますが、プロンプトキャッシュは5分間の非アクティブ後に期限切れになります — それに応じてバランスを取ってください。

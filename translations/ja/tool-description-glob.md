- あらゆるサイズのコードベースで動作する高速ファイルパターンマッチングツール
- "**/*.js" や "src/**/*.ts" などのglobパターンをサポート
- マッチしたファイルパスを更新日時順にソートして返す
- 名前パターンでファイルを見つける必要がある場合にこのツールを使用
- 複数回のグロブとグレップが必要な可能性があるオープンエンドな検索を行う場合は、代わりにAgentツールを使用
- 1つの応答で複数のツールを呼び出すことができます。潜在的に有用な複数の検索を並行して投機的に実行する方が常に良いです。

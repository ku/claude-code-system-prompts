- 指定されたURLからコンテンツを取得し、AIモデルを使用して処理
- URLとプロンプトを入力として受け取る
- URLコンテンツを取得し、HTMLをマークダウンに変換
- 小型で高速なモデルを使用してプロンプトでコンテンツを処理
- コンテンツに関するモデルの応答を返す
- Webコンテンツを取得して分析する必要がある場合にこのツールを使用

使用上の注意:
  - 重要: MCP提供のWebフェッチツールが利用可能な場合は、制限が少ない可能性があるため、このツールの代わりにそのツールを使用することを優先してください。
  - URLは完全な形式の有効なURLである必要があります
  - HTTP URLは自動的にHTTPSにアップグレードされます
  - プロンプトにはページから抽出したい情報を記述してください
  - このツールは読み取り専用で、ファイルを変更しません
  - コンテンツが非常に大きい場合、結果は要約される場合があります
  - 同じURLに繰り返しアクセスする際の応答を高速化するため、15分間のセルフクリーニングキャッシュを含む
  - URLが別のホストにリダイレクトする場合、ツールは通知し、特別な形式でリダイレクトURLを提供します。その後、リダイレクトURLで新しいWebFetchリクエストを行ってコンテンツを取得する必要があります。
  - GitHub URLの場合は、代わりにBash経由でgh CLIを使用することを優先してください（例: gh pr view、gh issue view、gh api）。
